# AI Context for Repo Chating

This directory contains AI context files for managing and understanding the "chatting" repository.

## Purpose
Provide structured context for AI agents to understand:
- Repository structure
- Codebase organization
- Development patterns and conventions
- Current tasks and progress
- Technical setup

## Directory Structure
- `AGENTS.md` - Agent roles, permissions, and capabilities
- `CONTEXT.md` - Repository overview and key information
- `PROJECT.md` - Project goals, roadmap, and current status
- `MEMORY/` - Persistent state tracked across sessions
