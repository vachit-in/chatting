# Nearby Chat App – Dual‑Mode Messaging & More

## 🚀 Overview
This new version of the app expands on the original **Nearby Chat** platform, adding a rich set of communication, entertainment and productivity features while keeping privacy at its core. The core concept remains: users can chat *permanently* with friends using a verified username (blue‑tick), or discover and talk to strangers *anonymously* for up to 24 hours.

---

## 📱 Core Functionality

### 1️⃣ Chatting
| Feature | Permanent Mode | Anonymous Mode |
|---------|----------------|----------------|
| **Identity** | Verified username + optional blue‑tick (ID verified) | Temporary anonymous ID & profile summary |
| **Registration** | Phone number + ID verification | None – create a temporary session ID |
| **Finding Users** | Search by username, contacts or global list | View nearby users via cell‑tower triangulation (no GPS)
| **Chat Duration** | Unlimited | 24 h temporary chat; can be extended to a lifetime *anonymous* friendship
| **Privacy** | Real identity visible only to chosen contacts | Identity never revealed unless user shares it voluntarily |
|
#### Features in Chat
- End‑to‑end encryption for all messages, calls and media.
- Custom emoji & sticker packs.
- File sharing up to 25 GB (cloud backed).
- Built‑in chat summarisation.
- Voice & video calling.
- Channels / group chats (work‑around style – create a dedicated room with a unique link).
- Event creation & RSVP support.
- Bill splitting and shared expenses.

### 2️⃣ Offline Games
| Game Type | Examples |
|-----------|----------|
| **Single‑player** | Candy Crush, Chess, Ludo, Subway Surfers, Card games |
| **Multiplayer** | Mini Militia, Ludo, Chess, Card games (peer‑to‑peer) |

### 3️⃣ Music Streaming in Chat
- Connect to Spotify, YouTube Music and Apple Music.
- Stream music directly within chat rooms or private conversations.

### 4️⃣ Bot Marketplace
- Developers can publish AI‑powered bot agents.
- Monetise bots via in‑app purchases or subscription.
- Bots can answer questions, play games, manage schedules, etc.

### 5️⃣ Nearby File Sharing
- Pair devices that are physically close (Wi‑Fi Direct / Bluetooth).
- Transfer files instantly without cloud upload.
