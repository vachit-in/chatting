# Nearby Chat App - Dual Mode

A comprehensive real-time messaging application that combines **Anonymous Nearby Chatting** with **Permanent Connected Chatting** - giving you the freedom to choose how you want to connect.

---

## 🌟 Feature Overview

### **Mode 1: Anonymous Nearby Chat** (Privacy-First)
### **Mode 2: Permanent Username Chat** (Identity-Based)

Users can freely switch between both modes at any time, enjoying the best of both worlds!

---

## 🕵️‍♂️ MODE 1: Anonymous Nearby Chat

A privacy-focused real-time messaging application that connects you with other nearby users anonymously - you'll NEVER reveal your identity.

### **How It Works:**

#### 1️⃣ **Create Anonymous Profile**
- Open the app (no registration/login needed)
- Set basic preferences (age range, interests, languages, location radius)
- You get a **temporary session ID** - this is your ONLY identity
- **NO phone number, username, name, or personal info required**

#### 2️⃣ **Find Nearby Users via Cell Tower**
- Browse anonymous profiles in your **cell tower coverage area**
- Works without GPS - uses cell tower triangulation
- Set your search radius (e.g., 1km, 5km, 10km)
- See who's nearby (all profiles look identical - no personal info)

#### 3️⃣ **Send Request Blindly**
- Choose any anonymous profile and send a chat request
- **You don't know who they are** - they could be anyone
- Request contains only your temporary session ID

#### 4️⃣ **Recipient Reviews Summary Only**
The receiving user sees **ONLY ANONYMOUS INFO**:
- ⚪ Temporary session ID (looks like random code)
- 📍 Location area (e.g., "Downtown Area" - never specific address)
- 🎯 Age range (from your anonymous profile, e.g., "25-35")
- 💫 Interests (e.g., "travel, music, tech" - whatever you set)
- 📸 Profile photo (if you chose to add - anonymous only)
- 🎨 Brief bio (fake/anon description)

**THEY NEVER SEE YOUR REAL IDENTITY** - Not even if they accept!

#### 5️⃣ **24-Hour Anonymous Chat Session**
Once both users accept:
- Enter a **24-hour temporary conversation**
- Unlimited text messages, voice messages, images, videos
- Voice and video calls supported
- End-to-end encrypted
- **Auto-deleted after 24 hours** - no digital footprint
- You both remain **completely anonymous** the entire time

#### 6️⃣ **Convert to Permanent Friends (Optional)**
- Before the 24 hours expire, either party can request to "convert"
- If **BOTH agree**, you become **permanent anonymous friends**
- Continue chatting anonymously indefinitely
- **STILL NO IDENTITY REVEALED** - just permanent anonymous chat

#### 7️⃣ **Automatic Cleanup**
- If conversation ends after 24 hours without conversion
- All messages, media, calls are permanently deleted
- No one knows they existed

### **Privacy Features:**
- 🔒 **Zero-Knowledge**: Your real identity is NEVER shared or stored
- 🗑️ **Ephemeral Data**: Everything auto-deleted after sessions
- 🔐 **End-to-End Encryption**: Messages fully secured
- 🚫 **No Logs**: We don't track users or conversations
- 🛡️ **Privacy Controls**: Block anyone, delete chats instantly

## 👤 MODE 2: Permanent Username Chat

A traditional real-time messaging app where you use your **real permanent username** - like WhatsApp, Signal, or Telegram.

### **How It Works:**

#### 1️⃣ **Create Account**
- Register with your **permanent username** (choose your own)
- Link account to **phone number** for verification
- Optional: Two-factor authentication for extra security
- Profile setup: Real name, photo, bio, status

#### 2️⃣ **Add Contacts**
- Import contacts from phone
- Search by username or phone number
- Request to add friends
- Accept/decline friend requests
- One-time chat or permanent connections

#### 3️⃣ **Direct Messaging**
- Send unlimited messages to your contacts
- Private group chats with friends
- Share photos, videos, files
- Voice and video calls
- End-to-end encrypted
- **Your real identity is known to contacts**

#### 4️⃣ **Features Available:**
- 💬 Text messaging
- 📸 Photo/Video sharing
- 🎤 Voice messages
- 📞 Voice calls
- 🎥 Video calls
- 📁 File transfer
- 📅 Scheduling & reminders
- 🔔 Push notifications (messages, calls)
- 📜 Chat history (backed up)
- 🎭 Disappearing messages (optional)
- 🔒 Message lock (E2E security)

### **Security Features:**
- 🔐 **End-to-End Encryption**: All chats & calls secured
- 🔐 **Secure Keys**: You control your encryption keys
- 🔒 **Two-Factor Auth**: Extra account protection
- 🛡️ **Anti-Hacking**: Built-in security measures
- 📦 **Server Backup**: Chat history saved (if chosen)

### **Privacy Options:**
- 🔒 **Private**: Full E2E encryption
- 📢 **Group Chats**: Multi-user conversations
- 📍 **Location Sharing**: Share with trusted contacts
- 📹 **Video Calls**: Face-to-face communication
- 🎤 **Voice Calls**: Audio communication
- 📵 **Read Receipts**: See when messages are read
- ⏰ **Message Status**: Sent, delivered, read
- 🚫 **Anti-Spam**: Report and block

## 🔄 SWITCHING BETWEEN MODES

You can **freely switch** between Anonymous and Permanent modes:

### **From Anonymous → Permanent:**
- Complete your 24-hour anonymous chat session
- Request to convert to permanent friendship
- If both agree, you now have a **permanent username connection**
- Switch to Permanent Username mode with that contact
- Your identity becomes known from this point

### **From Permanent → Anonymous:**
- Start fresh in Anonymous mode anytime
- Create new anonymous profile
- New temporary session ID
- Start completely anonymous chatting
- **Previous permanent chats remain separate**
